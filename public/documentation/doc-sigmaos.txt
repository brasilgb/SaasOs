1. Introduçăo
O Sistema de Gestăo de Ordens de Serviço foi desenvolvido para auxiliar empresas que realizam atendimentos técnicos, manutençăo, assistęncia técnica ou serviços externos.
A plataforma centraliza as principais operaçőes da empresa em um único ambiente, permitindo organizar:
• clientes
• ordens de serviço
• agendamentos
• peças e produtos
• comunicaçăo da equipe
• relatórios operacionais e financeiros
Com isso, o sistema proporciona maior organizaçăo, controle e produtividade no gerenciamento das atividades.
* O sistema foi desenvolvido com interface responsiva, permitindo sua utilizaçăo tanto em computadores quanto em dispositivos móveis, como celulares e tablets. Dessa forma, gestores e técnicos podem acessar as informaçőes do sistema de qualquer lugar, facilitando o acompanhamento de serviços e a gestăo das atividades da empresa.

2. Visăo Geral do Sistema
O sistema é dividido em módulos que organizam as funcionalidades de acordo com os processos operacionais da empresa.
Principais módulos disponíveis:
• Dashboard com indicadores
• Gestăo de clientes
• Ordens de serviço
• Orçamentos
• Agendamentos de visitas
• Mensagens internas da equipe
• Cadastro de peças e produtos
• Relatórios operacionais
• Configuraçőes do sistema
• Gestăo de usuários
Além disso, o sistema também oferece recursos complementares como:
• aplicativo para envio de imagens
• área de acompanhamento para clientes
• importaçăo de clientes por arquivo
• integraçăo com WhatsApp
• controle de vendas de produtos

3. Primeiros Passos (Configuraçăo Inicial Recomendada)
Antes de iniciar o uso do sistema no dia a dia da empresa, recomendamos realizar algumas configuraçőes iniciais.
Essas configuraçőes garantem o correto funcionamento das ordens de serviço, impressőes e mensagens automáticas.
Acesse o menu Configuraçőes e realize os seguintes cadastros.
3.1 Dados da empresa
Informe os dados básicos da empresa.
Essas informaçőes serăo utilizadas nos documentos gerados pelo sistema, como:
    • recibos
    • orçamentos
    • comprovantes
    • documentos impressos Dados recomendados:
    • nome da empresa
    • telefone
    • endereço
    • outras informaçőes institucionais

3.2 Cadastro de tipos de equipamento
O cadastro de tipos de equipamento facilita o registro das ordens de serviço.
Exemplos:
• celular
• notebook
• televisăo
• impressora
• tablet
• outros equipamentos atendidos pela empresa

3.3 Configuraçőes adicionais
No menu configuraçőes também podem ser definidos:
• mensagens padrăo para envio via WhatsApp
• textos utilizados em impressőes
• configuraçőes de etiquetas
• checklists de conferęncia de equipamentos
Essas opçőes podem ser ajustadas conforme o fluxo de trabalho da empresa.

4. Dashboard
O Dashboard apresenta uma visăo geral das atividades da empresa.
Nesta tela săo exibidos indicadores (KPIs) e gráficos com dados operacionais e financeiros.
Informaçőes que podem ser exibidas:
• desempenho operacional
• indicadores financeiros
• evoluçăo das atividades
• informaçőes recentes do sistema
Caso o módulo de vendas esteja habilitado, será exibido também um ícone de carrinho de compras para acesso rápido ao registro de vendas.

5. Gestăo de Clientes
O módulo de clientes permite registrar e administrar as informaçőes dos clientes da empresa.
Funcionalidades disponíveis
• cadastrar novos clientes
• editar informaçőes
• excluir registros
• visualizar histórico de serviços
• visualizar agendamentos
• enviar mensagens via WhatsApp
Açőes disponíveis dentro do cadastro do cliente
Em cada cliente é possível:
• visualizar ordens de serviço vinculadas
• visualizar agendamentos
• enviar mensagem via WhatsApp
• editar cadastro
• excluir registro

6. Ordens de Serviço
As ordens de serviço registram os atendimentos realizados pela empresa.
Cada ordem pode conter informaçőes como:
• cliente
• tipo de equipamento
• descriçăo do problema
• diagnóstico
• peças utilizadas
• status do serviço
Funcionalidades disponíveis
• criar ordens de serviço
• editar ordens existentes
• excluir ordens
• enviar mensagens ao cliente via WhatsApp
• registrar imagens do equipamento
• imprimir documentos relacionados

Documentos disponíveis para impressăo
O sistema permite gerar documentos importantes relacionados ao atendimento:
• recibo de entrada do equipamento
• recibo de entrega
• orçamento
• checklist de conferęncia

Registro de imagens
Durante o atendimento é possível registrar imagens do equipamento.
Essas imagens podem ser enviadas através do aplicativo auxiliar e ficam vinculadas ŕ ordem de serviço.

7. Orçamentos
O módulo de orçamentos permite gerar propostas para os clientes.
Funcionalidades disponíveis:
• criaçăo de orçamento
• geraçăo de orçamento em PDF
• ediçăo de orçamento
• exclusăo de orçamento

8. Agendamentos de Visitas
O módulo de agendamentos organiza as visitas técnicas.
Funçőes disponíveis
• criar agendamentos
• editar agendamentos
• excluir agendamentos
• enviar mensagem ao técnico via WhatsApp

9. Área do Cliente (Acompanhamento Online)
O sistema possui uma área exclusiva para acompanhamento do andamento dos serviços pelos clientes.
Após o cadastro da ordem de serviço, o sistema permite enviar ao cliente um link de acompanhamento através do WhatsApp.
Ao acessar esse link, o cliente poderá visualizar o andamento do atendimento diretamente pelo navegador.
Essa página funciona normalmente em computadores e também em celulares.
Informaçőes disponíveis para o cliente
O cliente poderá acompanhar:
• status atual do serviço
• atualizaçőes da ordem
• informaçőes do atendimento
Esse recurso aumenta a transparęncia e reduz a necessidade de contato frequente para solicitar atualizaçőes do serviço.

10. Mensagens Internas da Equipe
O sistema possui um módulo de comunicaçăo interna entre usuários.
Funcionamento:
• o remetente pode editar ou excluir a mensagem
• o destinatário pode visualizar e marcar a mensagem como lida
Esse recurso facilita a comunicaçăo da equipe dentro do próprio sistema.

11. Cadastro de Peças e Produtos
O sistema permite registrar itens utilizados nos serviços ou vendidos pela empresa.
Durante o cadastro é possível definir o tipo do item:
• peça
• produto
Funçőes disponíveis
• cadastro de itens
• ediçăo de registros
• exclusăo de registros

12. Controle de Estoque
O sistema possui controle de estoque para peças e produtos cadastrados.
Esse controle permite acompanhar a movimentaçăo dos itens utilizados nos serviços ou vendidos.

13. Relatórios do Sistema
O sistema disponibiliza diversos relatórios que auxiliam na análise das atividades da empresa.
Os relatórios podem ser filtrados por intervalo de datas.
Relatórios disponíveis:
• produtividade técnica
• ordens de serviço
• clientes
• agendamentos
• vendas de peças e produtos

14. Configuraçőes Gerais
O menu de configuraçőes centraliza diversas opçőes administrativas do sistema.
Entre as configuraçőes disponíveis estăo:
• dados da empresa
• mensagens padrăo para WhatsApp
• textos utilizados em documentos
• etiquetas
• tipos de equipamento
• checklists
• outras configuraçőes administrativas

15. Gestăo de Usuários
O sistema permite o cadastro e gerenciamento de usuários.
Funçőes disponíveis:
• cadastro de usuários
• ediçăo de usuários
• exclusăo de usuários

16. Aplicativo para Upload de Imagens
O sistema possui um aplicativo auxiliar que permite registrar e enviar imagens relacionadas ŕs ordens de serviço.
As imagens enviadas ficam vinculadas ŕ ordem correspondente no sistema.
Esse recurso é útil para registrar:
• estado do equipamento
• diagnóstico
• evidęncias do serviço realizado

17. Importaçăo de Clientes por CSV
Para facilitar o cadastro inicial de clientes, o sistema permite importar registros utilizando arquivos 
CSV.
Essa funcionalidade possibilita inserir diversos clientes de uma única vez.

18. Módulo de Vendas
O sistema possui um módulo opcional para registro de vendas de produtos.
Esse módulo pode ser habilitado nas configuraçőes do sistema.
Após habilitado:
    • aparecerá um ícone de carrinho no dashboard
    • será exibido o menu Vendas Nesse módulo será possível:
    • registrar vendas
    • visualizar vendas realizadas
    • imprimir recibos ou tickets
    • cancelar vendas registradas

19. Emissăo de Notas Fiscais
O sistema năo realiza emissăo automática de notas fiscais.
Entretanto, o sistema disponibiliza botőes que direcionam para os sistemas de emissăo utilizados pela empresa.
Para serviços pode ser utilizada a emissăo de NFSe.
Para vendas de produtos pode ser utilizada a emissăo de NFe estadual.
Para facilitar o preenchimento da nota fiscal, o sistema apresenta um resumo das informaçőes necessárias.
Esse resumo pode ser visualizado:
• no topo da tela de ediçăo da ordem de serviço
• na tabela inicial de listagem das ordens
Após a conclusăo do serviço, podem ser utilizados os seguintes status para acompanhamento:
• serviço concluído
• cliente avisado
• entregue ao cliente

20. Boas Práticas de Uso do Sistema
Para melhor organizaçăo das atividades da empresa recomendamos:
• manter os dados dos clientes sempre atualizados
• registrar corretamente as ordens de serviço
• atualizar o status das ordens conforme o andamento do atendimento
• utilizar os relatórios para acompanhar a produtividade da equipe
